This folder contains the wiring table, click [here](https://app.cirkitdesigner.com/project/13720d27-29c6-4cf1-b9f1-eaf993293925) to edit the wiring yourself using cirkit.

Wiring Table:
```
OLED GND  -> ESP32 GND
OLED VCC  -> ESP32 3.3V
OLED SCL  -> ESP32 GPIO 22
OLED SDA  -> ESP32 GPIO 21

Sensor GND -> ESP32 GND
Sensor VCC -> ESP32 3.3V
Sensor NC  -> not connected
Sensor SIG -> ESP32 GPIO 4
```
